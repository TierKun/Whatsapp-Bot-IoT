## Windows

* [`Download Node JS`](https://nodejs.org/en/download/)

* [`Download Git`](https://git-scm.com/download/win)





## Cloning this repo

```cmd

> git clone https://github.com/TierKun/Telegram-Bot-IoT

> cd Telegram-Bot-IoT

```



## Install the package

```cmd

> npm i

```
